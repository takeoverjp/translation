# HTTP Live Streaming
## draft-pantos-http-live-streaming-12

### 概要

この仕様書では、終端のないマルチメディアストリームを転送するためのプロトコルについて述べる。このプロトコルは、転送するファイルの形式と、ストリームのサーバ（送信者）とクライアント（受信者）の振る舞いについて規定する。この仕様書では*バージョン６*のプロトコルについて解説する。
